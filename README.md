## Принцип работы
Сначала мы импортируем Tkinter. Создаем окно и убираем возможность изменять размер окна, а также добавляем заголовок.
```py
from tkinter import *

root = Tk()
root.resizable(width=False, height=False)
root.title("Калькулятор")
```
